How about an Android app made with Python?

Tired of having to save a new contact every single time that a given Seller needs to contact a new prospect, that person came to me to ask for a method to start a new WhatsApp conversation without saving the contact on her phone. That's when NoSave Android App came to live. Using WhatsApps Api to start a new conversation based in the number alone this apps gives free time with no need of save a new contact every time you need to make a call over WhatsApp. Cool.

As a part of this project between my duties were:

  • Designed the connection-script
  • Designed the app´s user interface using kivy module
